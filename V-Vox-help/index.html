- About -

1.What is VancedVoxiom?

VancedVoxiom is an extension for voxiom.io.
It is a multifunctional and secure extension that can be customized in many ways.

2.What can we do with it?

Add custom crosshairs.
Change logo and background image.
Loading CSS.
More features will be added in the future!

- Help -

0. Reset settings

1.Press F12 and open console.
2.type localStorage.removeItem("KeyName");
KeyNames values
├ checkBoxSaved true / false
├ crosshairUrlSaved URL
├ crosshairHeightSaved Num
├ crosshairWidthSaved Num
├ logoUrlSaved URL
├ logoTextSaved TEXT
├ bgStyleSaved image / color
├ bgUrlSaved URL
├ colorDegSaved Num
├ color1Saved #HEX
├ color2Saved #HEX
├ color3Saved #HEX
└ cssValSaved text

1. Crosshair

URL : Enter Crosshair's image URL.
W : Width of crosshair
H: Height of crosshair
Don't forget to set the crosshair style to None from the in-game settings!
You don't need crosshair? Set W and H to 0.
Auto Default Size : Set to the original size of the input image.

2. Logo and text

URL : Enter logo image URL.
Recommended image size is 813*197
TEXT : You can change the text under the logo.
Image Entering a space instead of a blank space hides it

3-1. Background Image

URL : Enter background image URL
You can reset the image by leaving the URL input field blank

3-2. Background Color

Deg : Change the angle of the gradient
Color1 : Gradient color 1
Color2 : Gradient color 2
Color3 : Gradient color 3

4. CSS Loader

You can input css directly and reflect it in real time.
Don't know how to write CSS?
Google it. It's very easy.
Let me Google it!
