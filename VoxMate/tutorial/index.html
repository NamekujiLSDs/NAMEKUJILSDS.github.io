Crosshair

You can apply any crosshair image of your choice.
If necessary, switch the in-game crosshair setting to “NONE”.

Enable Custom Crosshair

Turn on/off custom crosshair display.
It is enabled by default.

Crosshair Type

・URL
Load an crosshair image from a URL.
png,gif is recommended.
・Local
Load an crosshair image from local file.
・List
Select crosshair image from a crosshair folder in swapper folder.
More info in #Swapper.

Crosshair URL

Set the URL for the crosshair.
It is recommended to use imgur or similar.

Local Crosshair File

Locally stored crosshair files can be selected.
Only png,gif,apng can be used

Local Crosshair List

You can select crosshairs in the Swapper folder.
More info in #Swapper.

Auto Set Size

Obtains the native size of the image used for crosshairs and sets it automatically

Crosshair Width

Sets the width of the crosshairs.
1px-1024px

Crosshair Height

Sets the height of the crosshairs.
1px-1024px

Crosshair Opacity

Sets the transparency of the crosshairs.
0 is completely transparent and 1 is opaque.

Crosshair Rendering Method

Sets the algorithm for scaling images when displaying crosshair images.
・Pixelated
The image is scaled with the "nearest neighbor" or similar algorithm to the nearest integer multiple of the original image size, then uses smooth interpolation to bring the image to the final desired size. This is intended to preserve a "pixelated" look without introducing scaling artifacts when the upscaled resolution isn't an integer multiple of the original.

・Crisp Edges
The image is scaled with an algorithm such as "nearest neighbor" that preserves contrast and edges in the image. Generally intended for images such as pixel art or line drawings, no blurring or color smoothing occurs.

CSS

User-created CSS can be reflected

Enable Custom CSS

Turning custom CSS on and off.

CSS Type

・URL
You can load CSS from a URL.
You must have a link to CSS that is compatible with CORS (such as those published via GithubPages.)
・Local File
Load locally stored CSS.
You must select local CSS from the “Local CSS File” section.
・List
You can select CSS from the list of CSS files placed in the swapper folder.
More info in #Swapper.

Swapper

Explanation of resource swappers and other resources.
Please ask questions on Discord if you have difficulty understanding any part.
*Basically, the client needs to be restarted.

About Swapper Folder

The swapper folder is located in documents/vmc-swap
You can also open it by pressing Windows + R and enter %userprofile%\documents\vmc-swap

About CSS and Crosshair List
If you have run VoxMate at least once, the folders “css” and “crosshair” are created in the swapper folder.
(If they do not exist, please create them.)
Place your CSS and image files in those folders and they will appear in the “List”.

Use Default Swapper List

Use the swap list that is included by default.
Default listing details: open_in_new swapper-default.json

Use User Swapper List

Enable User-created swapper list.
The swapper list must be named swapper-user.json and located in the C:\Users\{username}\Documents\vmc-swap folder.
The structure of the list should be as follows: 
{
    "https://voxiom.io/example.jpg" : "FileName1.jpg",
    "https://voxiom.io/sound.ogg" : "FileName2.ogg", 
    "https://voxiom.io/logo.png" : "Folder/FileName3.png"
}

If the user list and the default list contain the same URL and both are valid, the user list takes precedence.

Ad Blocker

Block ads.

About Ad Blocker

Adblocker blocks all those requests if the URLs specified in the block list are included in the web request.
If you are trying to work with something you do not know what you are doing, DO NOT TOUCH IT.

Use Default Block List

Use the block list that is included by default.
Default list details: open_in_new adblock-default.json

Use User Block List

User user-created adblock list,
The name of the block list must be adblock-user.json and must be placed directly under C:\Users\{username}\Documents\vmc-swap.
The structure of the list should be as follows: 

{
    "urls":[
        "example.com",
        "ads.com",
        "adserver.com
    ]
}
