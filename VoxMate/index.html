Client Safety

To ensure client safety and transparency, we do the following
  • Using Electron
    • Electron is a platform that combines Chromium and Node.js and includes industry standard security features.
    • Sandboxing the web page execution environment, restricting and controlling access to the system.
    • The web page eTxecution environment and the system execution environment are separated. m
  • Open Source Transparency
    • All code (except secret key) is available on the Github project.
    • Anyone can review the code and verify that it is secure enough.
    • All libraries use reliable open source libraries.
  • User Data Protection
    • All user data is stored locally and is not accessed remotely by the developer.
    • I am not acquiring any user data.
  • Features to improve user safety
    • Automatic updates can be retrieved and patches applied at startup.
    • Sandboxing to isolate and process unknown files and data.
    • Thorough code review to prevent bugs and vulnerabilities.
