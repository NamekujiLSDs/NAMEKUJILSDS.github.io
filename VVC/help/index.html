Vanced Voxiom Client Help Page
Client
Keybinds
- F1 : Open client menu.
- F5 : Reload page.
- F6 : Use quick join.
- F8 : Open match list.
- F11 : Toggle fullscreen.
- F12 : Open Devtools.
Menu
Custom background image
The background of the menu screen can be changed.
Use the URL of the image you want to use for the background.
If you want to use an offline file, convert it to BASE64 or use imgur or Discord etc.
Custom game logo image
You can change the "VOXIOM" logo on the menu screen.
Please insert the URL of the image you want to use
Custom logo text
You can edit the text written under the logo on the menu screen.
Crosshair
Use custom crosshair
Check the box to use custom crosshairs.
Custom crosshair URL
Insert the image URL for the custom crosshair; GIF, PNG, etc. can be used.
crosshair Size X
You can set the width of the crosshair. The unit is px.
crosshair Size Y
You can set the height of the crosshair. The unit is px.
Auto crosshair sizing
Match the size of the crosshairs to the original size of the image.
CSS
Custom CSS type
- Disable : Disable custom CSS.
- Textarea : Use the CSS written in the CSS Textarea.
- Local File : Use the CSS selected in Local CSS.
- From URL : Use CSS from the URL specified by CSS url.
CSS Textarea
Paste the CSS to be used.
Local CSS
Select the CSS to be used from a local file.
CSS url
Enter the URL of the CSS to be used.
Only specific URLs can be used.
(e.g. https://example.github.io/style.css)
QuickJoin
Region
Select the region you want to enter with Quick Join
Gamemode
Select the gamemode you want to enter with Quick Join
Mini Tool
Disable Snowflake
Disable the snow effects added in the Christmas update.
Disable FreeGem popup
Disable the free gem popup that comes up once an hour when you are logged into your account. (You will not receive any Free Gems.)
Enable chat to webhook
Sends all chat content to the webhook.
Webhook URL
Set the URL for Chat to Webhook.

How to use Resource Swapper
What is Resource Swapper?
The resource swapper can change the sounds in the Voxiom and change the player skin.
How to use Resource Swapper
There is a folder named VVC-Swapper in Documents folder (C:\Users\USERNAME\Documents).
You can swap by putting a file with the corresponding name in it.

How to use additional swap rule?
Join the official Discord server and check the #share-json-settings channel for pinned posts.
Download the swapperConfig.json in that post and put it in your swapper folder.
You may edit this JSON according to the following writing rules "URL": "FILENAME.ext"
How to use Request Rejecter
What is Request Rejecter
This client still has the ability to block web requests, although it is still WIP.
This feature can be used well to block various advertisements.
How to use Request Rejecter
Join the official Discord server and check the #share-json-settings channel for pinned posts.
Download the rejectConfig.json in that post and put it in your swapper folder.
You can edit the JSON file as follows

{ "reject":[
"STRING OR URL",
]}

For example, entering "voxiom.io" will block all URLs containing string "voxiom.io".
If you type "A", all URLs containing string "A" will be blocked.
We recommend writing something like "://example.com/libs"
Use with caution!
