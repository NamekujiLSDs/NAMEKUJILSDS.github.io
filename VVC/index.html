What is VVC?
VVC (VancedVoxiomClient) is a client for Voxiom.io that I am developing personally. It is an unofficial client with high performance and many features.

Why VVC?
VVC allows you to play with high functionality and unlimited FPS. This will help you raise your player skills to the next level.
Features
- Custom Crosshair
- Custom CSS
- Game overlay
- Match list
- Quick join
- Disable Freegem popups
- Skin changer
- Unlimited FPS
...And more!
VVC is safe?
When you ask this question you will not trust me to just say that "It is safe". This project is being developed in open code. I have no intention of being absolutely safe about this project. Please check the project source yourself and decide if it is safe or not.
