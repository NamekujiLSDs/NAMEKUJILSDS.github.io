What is Vanced Voxiom Client?
VancedVoxiomClient (VVC) is an unofficial client for Voxiom.io.
VVC can improve the performance and play experience of Voxiom.io. VVC also adds a variety of customization options to Voxiom.io.
This will allow you to change the look and features of your game.
What can VVC do?
- Change menu logo
- Change menu title
- Change background image
- Custom CSS injector
- Custom Crosshair
- Custom Game Overlay Images
- Optimized Performance
- Disabling FPS caps
- Disabling Gem Popups
- Disable event effects (e.g. snow)
- Automatic updates
